# API REST .NET 8 com SQL Server Express

## Descrição
Este projeto é uma API REST desenvolvida em .NET 8 com um banco de dados SQL Server Express. A API permite realizar operações CRUD (Create, Read, Update, Delete) em uma base de dados.

## Pré-requisitos
- [Visual Studio 2022 ou superior](https://visualstudio.microsoft.com/)
- [.NET SDK 8.0 ou superior](https://dotnet.microsoft.com/download)
- [SQL Server Express](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
- [Git](https://git-scm.com/)
